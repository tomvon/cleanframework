Live Sidebar Example

This page demonstrates a real-world implementation of the sidebar component. The sidebar you see on the left is fully functional and includes:

  • Brand section - "Admin Panel" with icon and built-in toggle button
  • Navigation sections - Organized menu items grouped by category
  • Badges - Notification counts and status indicators
  • User profile - Fixed at the bottom with avatar and role
  • Compact mode - Click the toggle button in the sidebar header to switch modes

Implementation

To use the sidebar in your project:

  1. Add class="has-sidebar" to your body tag
  2. Include the sidebar component after your header
  3. Wrap your main content in a <div class="main-content">
  4. The sidebar will automatically adjust for mobile devices
