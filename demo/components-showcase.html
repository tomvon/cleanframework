FAQ

What is Clean Framework?

Clean Framework is a semantic CSS framework that prioritizes readable HTML with intuitive class names. It provides essential UI components without the complexity of utility classes.

How do I get started?

Simply include the CSS file in your HTML and start using semantic component classes like .card, .hero, and .nav. No build process or configuration required.

Is JavaScript required?

No, JavaScript is optional. The framework provides progressive enhancement - components work without JS but gain additional functionality when JS is included.

Can I customize the styles?

Yes, the framework uses CSS custom properties for theming and provides SCSS source files for deeper customization. You can modify colors, spacing, and component styles to match your brand.
