Getting Started

Installation

There are several ways to include Clean Framework in your project:

CDN

<!-- Clean Framework CSS -->
<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/tomvon/cleanframework@latest/clean-framework.css">

<!-- Clean Framework JavaScript -->
<script src="https://cdn.jsdelivr.net/gh/tomvon/cleanframework@latest/clean-framework.js"></script>

Download

Download the files directly and include them in your project:

Download CSS Download JS

Quick Start

Start with this basic HTML template:

<!DOCTYPE html>
<html lang="en" data-theme="light">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My Clean Framework Site</title>
    <link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/tomvon/cleanframework@latest/clean-framework.css">
</head>
<body>
    <nav class="cf-nav">
        <div class="cf-container">
            <div class="cf-nav-container">
                <a href="#" class="cf-nav-brand">My Site</a>
                <ul class="cf-nav-menu">
                    <li><a href="#" class="cf-nav-link">Home</a></li>
                    <li><a href="#" class="cf-nav-link">About</a></li>
                </ul>
            </div>
        </div>
    </nav>

    <section class="cf-hero">
        <div class="cf-container">
            <h1 class="cf-hero-title">Welcome to Clean Framework</h1>
            <p class="cf-hero-subtitle">Build beautiful, semantic websites</p>
        </div>
    </section>

    <script src="https://cdn.jsdelivr.net/gh/tomvon/cleanframework@latest/clean-framework.js"></script>
</body>
</html>

Customization

Clean Framework uses CSS custom properties for easy customization. Override any of these variables in your CSS:

:root {
    /* Colors */
    --cf-primary: #667eea;
    --cf-secondary: #48bb78;
    --cf-accent: #ed8936;
    
    /* Typography */
    --cf-font-sans: -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, sans-serif;
    --cf-font-size-base: 16px;
    
    /* Spacing */
    --cf-spacing-xs: 0.25rem;
    --cf-spacing-sm: 0.5rem;
    --cf-spacing-md: 1rem;
    --cf-spacing-lg: 1.5rem;
    --cf-spacing-xl: 2rem;
    
    /* Borders */
    --cf-border-radius: 8px;
    --cf-border-width: 1px;
}

Dark Mode

Clean Framework includes built-in dark mode support:

<!-- Enable dark mode -->
<html lang="en" data-theme="dark">

<!-- Toggle dark mode with JavaScript -->
<script>
    // Toggle dark mode
    function toggleDarkMode() {
        const html = document.documentElement;
        const currentTheme = html.getAttribute('data-theme');
        const newTheme = currentTheme === 'dark' ? 'light' : 'dark';
        html.setAttribute('data-theme', newTheme);
    }
</script>

Dark Mode Variables

All colors automatically adjust in dark mode. You can customize dark mode colors:

[data-theme="dark"] {
    --cf-bg-primary: #1a202c;
    --cf-bg-secondary: #2d3748;
    --cf-text-primary: #f7fafc;
    --cf-text-secondary: #e2e8f0;
}

JavaScript API

Clean Framework includes JavaScript functions for interactive components:

Modal Functions

// Open a modal
openModal('modalId');

// Close a modal
closeModal('modalId');

// Toggle a modal
toggleModal('modalId');

Tab Functions

// Switch to a specific tab
switchTab(event, 'tabId');

// Initialize tabs (called automatically)
initTabs();

Accordion Functions

// Toggle accordion item
toggleAccordion(event);

// Open specific accordion
openAccordion('accordionId');

// Close all accordions
closeAllAccordions();

Navigation Functions

// Toggle mobile navigation
toggleNav();

// Close mobile navigation
closeNav();

Utility Functions

// Toggle dark mode
toggleDarkMode();

// Show notification
showNotification('message', 'type', duration);

// Smooth scroll to element
smoothScrollTo('elementId');
